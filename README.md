
# 📊 Análise de Dados Netflix

## 📋 Sobre o Projeto
Análise exploratória de dados da Netflix utilizando Python para ETL (Extract, Transform, Load) e Power BI para visualização. O projeto processa múltiplos datasets em Excel, realiza limpeza e transformação dos dados, e apresenta insights através de um dashboard interativo.

## 🎯 Objetivos
- Análise do catálogo de conteúdo da Netflix
- Identificação de tendências de conteúdo
- Visualização da distribuição de gêneros
- Análise temporal do crescimento do catálogo
- Métricas de popularidade de conteúdo

## 🛠️ Tecnologias Utilizadas
- **Python 3.13
  - Pandas (Manipulação de dados)
  - Numpy (Operações numéricas)
  - Visual Studio Code (documentação)
- **Power BI** (Visualização de dados)
- **Excel** (Datasets fonte)


## 🔄 Pipeline de Dados

### 1. Extração (Extract)
- Coleta de múltiplos arquivos Excel
- Identificação de fontes de dados relevantes
- Consolidação inicial dos datasets

### 2. Transformação (Transform)
- Limpeza de dados nulos e duplicados
- Padronização de formatos
- Correção de inconsistências
- Criação de métricas calculadas
- Enriquecimento de dados

### 3. Carregamento (Load)
- Exportação dos dados processados
- Integração com Power BI
- Criação de modelo de dados otimizado

## 📊 Dashboard Power BI
O dashboard inclui:
- Visão geral do catálogo
- Análise temporal de lançamentos
- Distribuição por gêneros
- Métricas de popularidade
- Análise geográfica
- KPIs principais


## 📈 Principais Insights
- Crescimento anual do catálogo
- Gêneros mais populares por região
- Padrões de lançamento de conteúdo
- Tendências de produção por país
- Análise de sucesso por categoria
